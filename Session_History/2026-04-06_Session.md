# Session — 2026-04-06 — Design Overhaul, Critique, and Bug Fixes

## Summary
Completed 6 design tasks (scroll reveals, feature verification, animation polish, design context, full UX critique with fixes), then fixed broken Booking.com hotel links across all 9 hotels.

## What Was Built/Changed

### Design Features (from prior context, completed this session)
- **Scroll-triggered section reveals**: IntersectionObserver with stagger delays, `.reveal` class system, re-observation on dynamic content
- **Evening mode toggle**: Day/evening toggle with localStorage persistence, CSS class switching, body/panel crossfade transitions
- **Restaurant wishlist upgrade**: Grouped by city with section headers
- **View fade transitions**: `fadeInView()` with reflow-forced animation on view switches
- **Password gate fade-out**: Smooth 0.4s opacity transition instead of instant hide

### Animation Polish
- Global color crossfade on evening toggle (body, topbar, tab-bar, panels)
- Map marker hover scale (1.15x with shadow)
- Map hint text ("Tap a marker for details")
- Shimmer pause on tab hidden (visibilitychange API)

### UX Critique Fixes (scored 25/40 → implemented all requested fixes)
- **Touch targets**: `.check-box` 20→24px, `.pack-box` 16→20px, `.bp-check` 18→22px, parent rows min-height 44px
- **Evening contrast**: 26 rules changed from `#8a9aaa` to `#9aacbe` (~4.7:1 ratio)
- **Evening toggle label**: Added "Evening" text label before toggle
- **Archive toggle**: Text changed to "Other ideas we considered" / "Hide alternatives"
- **Timeline bar**: `.tl-day` 10→13px, `.tl-label` 9→11px, bar height 56→64px
- **Transport notes**: Fixed `display:flex` wrapping bug with positioned bullet
- **Weather timeout**: 8s Promise.race timeout on fetch
- **Booking toggle performance**: Local `classList.toggle()` instead of full `renderPlan()` re-render
- **Duplicate class attributes**: Fixed `<div class="ov-cal-day" class="ov-cal-castle">` merge
- **Sunset shimmer battery**: Paused when tab hidden via visibilitychange
- **Mobile breakpoints**: 44px min tab buttons, 320px map, 400px timeline scroll

### Hotel Links Fix
- All 9 Booking.com hotel links (6 Porto + 3 Munich) switched from direct URL slugs to search-based URLs
- Direct slugs returned 404 "Page Not Found" on Booking.com
- New format: `booking.com/searchresults.html?ss=Hotel+Name` — reliably lands on correct hotel
